# Audio System

Application React minimale dediee au systeme audio.

## Demarrage

```bash
pnpm install
pnpm dev
```

## Scripts

- `npm run dev` lance l'application en local
- `npm run build` verifie TypeScript puis genere le build production
- `npm run lint` lance ESLint sur le projet
